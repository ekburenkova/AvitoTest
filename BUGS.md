### Отрицательное значение price проходит валидацию

Severity: Средняя

Окружение:
API: https://qa-internship.avito.com
Метод: POST /api/1/item

Шаги воспроизведения:
1. Отправить POST запрос с полем "price": -1
2. Указать валидные остальные поля

Фактический результат:
Объявление успешно создаётся

Ожидаемый результат:
Сервис должен вернуть ошибку (400 Bad Request) и не создавать объявление

Примечание:
Цена не может быть отрицательной согласно бизнес-логике
