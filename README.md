# В проекте собраны:

- автоматические тесты API (папка tests)
- список тест-кейсов (файл TESTCASES.md)
- найденные баги API (файл BUGS.md)
- найденные баги на интерфейсе (файл UI_BUGS.md)
- отчёт о выполнении тестов (папка allure-report)
- скриншоты отчёта Allure

# Установка и запуск:

1. Установите Python (если ещё не установлен)
2. Скачайте проект (через кнопку "Code" → "Download ZIP" или через git)
3. Откройте папку проекта в терминале
4. Создайте виртуальное окружение: python -m venv .venv
5. Активируйте его. Для Windows: .venv\Scripts\activate
6. Установите зависимости: pip install -r requirements.txt
7. Запустите тесты: pytest

# Просмотр отчёта Allure:

1. Выполните команду: pytest --alluredir=allure-results
2. Выполните команду: allure open allure-report

# Примечания:

1. Тесты выполняются на общем тестовом сервере. Из-за этого иногда могут возникать ошибки (например, статус-код 500), связанные с нагрузкой, а не с самими тестами.
2. Все найденные проблемы описаны в файлах: BUGS.md. Один из тестов намеренно падает — он показывает найденный баг (отрицательная цена принимается системой).
